## ✨ Overview

The **Blood Group Identification System** is a sophisticated desktop application designed for medical education and demonstration. It uses computer vision to detect blood agglutination (clumping) in Anti-A, Anti-B, and Anti-D (Rh) reagents, providing a high-confidence blood type determination.

Beyond simple analysis, the system integrates a full patient management suite and automated reporting.

---

## 🚀 Key Features

### 🔍 Advanced Image Processing
* **ROI Extraction**: Automatically isolates blood spots from slide images using HSV color masking.
* **Agglutination Scoring**: A proprietary 5-feature scoring engine:
    * `F1`: Internal fragment count
    * `F2`: Fragment area ratio
    * `F3`: Pixel intensity standard deviation
    * `F4`: Laplacian texture roughness
    * `F5`: Edge density analysis
* **Relative Scoring**: Graceful fallback and relative thresholding for varied lighting conditions.

### 👤 Patient Management
* **Secure Database**: Persistent storage for patient records and historical test results using SQLite.
* **Patient History**: Easily track and search previous test results by patient name.

### 📋 Reporting & Integration
* **Professional PDF Reports**: Automated generation of clinical-style reports including test scores, compatibility maps, and patient info.
* **Email Automation**: Directly send reports to doctors via SMTP (Gmail integration).
* **Statistical Insights**: Built-in data visualization for blood type distribution across the database.

## ⚙️ Installation & Setup

### 1. Clone the Repository
```bash
git clone https://github.com/Moubina-naz/Blood-Group-Detection.git
cd Blood-Group-Detection
```

### 2. Set Up Environment
```bash
# Create virtual environment
python -m venv venv

# Activate (Windows)
venv\Scripts\activate

# Activate (macOS/Linux)
source venv/bin/activate
```

### 3. Install Dependencies
```bash
pip install -r requirements.txt
```
*Required packages: `PyQt5`, `opencv-python`, `numpy`, `matplotlib`, `fpdf2`, `sqlite3`*

### 4. Run the Application
```bash
python app.py
```

## 🧪 Usage Guide

1.  **Identity**: Enter patient details (Name, Age, Gender, etc.).
2.  **Blood Test**: Upload clear images for Anti-A, Anti-B, and Anti-D test spots.
3.  **Analyze**: Click "Run Analysis". The system will compute agglutination scores (0-100). A score **> 40** is typically considered positive.
4.  **Report**: View final results, check compatibility, generate a PDF, or email the report directly.
